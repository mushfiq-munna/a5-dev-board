Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the Document Object Model (DOM). Here are some common methods-
By ID (getElementById), By Class Name (getElementsByClassName), By Tag Name (getElementsByTagName), By Name Attribute (getElementsByName),By CSS Selectors (querySelector), Using Traversal from Parent (parentElement.children).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property recognizes HTML tags and renders the content according to the tags. innerText and textContent ignore HTML tags and treat them as part of the text.

Question-3: What is event delegation in the DOM?

In DOM event delegation, a single event listener is attached to a parent element to handle events that bubble up from its child elements, instead of attaching listeners to each child individually.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process that propagates events from an element to its parent elements. It's a way to handle events in a browser, such as when a button is clicked.

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript provides multiple ways to dynamically manipulate the DOM by creating, adding, and removing elements.
To create an element, I can use the document.createElement() method. To add an element, I can use the appendChild(), append(), prepend() and insertBefore() method. To remove an element, I can use the removeChild() method.
